# Library Management System

A console-based Library Management System developed in C++ using Data Structures and File Handling.

## Features

- Add Book
- Update Book
- Delete Book
- Search Book
- Issue Book
- Return Book
- View Issued Books
- Issue History
- Multiple Book Copies
- File Handling
- Hash Table (unordered_map)
- Linked List (list)

## Technologies Used

- C++
- Data Structures
- Object-Oriented Programming
- File Handling
- STL

## Future Enhancements

- Binary Search Tree (BST)
- Fine Calculation
- Student Management
- Search by Author
- Search by Title
